caishau

[caishau nf., molaire en béarnais]

Interface RShiny pour le traitement des données d'analyses texturales de micro-usure dentaire.

  • Manipulation des données :
    • Selection et filtres
    • Moyennes et médianes
  • Outils graphiques :
    • Boxplots
    • Distribution
  • Statistiques :
    • Matrices de corrélation
    • Analyses de variance
    • Tests de normalité
  • Populations de réfénces intégrées
  • Jeu de données test

caishau est en développement depuis bien trop longtemps (2022) mais sera disponible ici.
